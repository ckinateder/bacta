# Arby's

A statistical arbitrage pairs trading bot prototype.

## Stack
- Python
- SimFin
- Alpaca

## Usage

### Development

My `.env` file has the following fields:

```env
SIMFIN_API_KEY=<api key here>
SIMFIN_DATA_DIR=data/simfin
DATA_DIR=data
```

Right now, most work is being done in `src/databuilder.py`.
